# LibraryProject
This is a Django project created as part of the ALX Django Learning Lab.

## Project Structure
- **manage.py**: Command-line utility for interacting with the Django project
- **LibraryProject/**: Main project directory containing:
  - **settings.py**: Configuration settings for the Django project
  - **urls.py**: URL declarations for the project
  - **wsgi.py**: WSGI-compatible web servers entry point
  - **asgi.py**: ASGI-compatible web servers entry point

## Running the Project
1. Navigate to the project directory
2. Run: `python manage.py runserver`
3. Open browser to: http://127.0.0.1:8000/
